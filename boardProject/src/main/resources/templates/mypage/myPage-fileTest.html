파일 업/다운로드 테스트

          [form 태그의 encType 속성]
  
          encType 은 무엇인가
          - 데이터를 서버로 전송할 때 데이터의 형태(인코딩)를 지정하는 속성
  
          1. enctype="application/x-www-form-urlencoded"
            - 기본값. URL 인코딩된 "문자열로" 서버에 제출
  
          2. enctype="multipart/form-data"(무조건 POST 방식에서만 동작)
            - 제출되는 여러 데이터 타입에 맞춰서 인코딩하여 제출
              -> 문자열 : String 
              -> 숫자 : String(Spring에서 자료형 바뀜)
              -> 파일 : 2진 데이터
  
          3. enctype="text/plain"
          - 텍스트로 서버에 제출함
